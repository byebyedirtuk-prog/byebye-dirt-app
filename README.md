# Bye Bye Dirt Cleaning

Base Next.js project for the Bye Bye Dirt Cleaning responsive web app.

## Stack

- Next.js App Router
- TypeScript

## Source Of Truth

Use `docs/ByeByeDirt_App_Specification_for_Codex.md`.

## Setup

1. Run `npm install`.
2. Run `npm run dev`.
3. Open `http://localhost:3000`.

Firebase is intentionally not configured in this setup phase.
